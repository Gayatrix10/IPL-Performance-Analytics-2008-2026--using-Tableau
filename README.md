# IPL Performance Analytics Dashboard (2008–2026)

## Project Overview

This project analyzes Indian Premier League (IPL) data using Tableau to generate insights into team performance, batting performance, and bowling performance.

The project consists of three interactive dashboards:

### 1. Executive Overview Dashboard

* Total Matches
* Total Runs
* Total Wickets
* Total Players
* Season-wise Trends
* Top Winning Teams
* Player of the Match Analysis

### 2. Batting Intelligence Dashboard

* Top Run Scorers
* Strike Rate vs Batting Average Analysis
* Boundary Analysis
* Runs by Match Phase
* Impact Player Analysis
* Season-wise Top Scorers

### 3. Bowling Intelligence Dashboard

* Top Wicket Takers
* Economy Rate vs Bowling Average
* Purple Cap Winners
* Wickets by Match Phase
* Bowling Style Analysis

## Tools Used

* Tableau Public
* Data Visualization
* Data Analytics
* KPI Design
* Interactive Dashboards
